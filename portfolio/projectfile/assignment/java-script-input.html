Check Leap Year in JavaScript Programming Language

Code (Prompt)

let year = prompt("Enter a year:"); let result = (year % 4 === 0 && (year % 100 !== 0 || year % 400 === 0)) ? `${year} is a leap year.` : `${year} is not a leap year.`; alert(result);

Output

Enter a year: 2000
2000 is a leap year.

Code (Assigned Value)

const year = 2024; // Assigned value if ((year % 4 === 0 && year % 100 !== 0) || (year % 400 === 0)) { alert(`${year} is a leap year.`); } else { alert(`${year} is not a leap year.`); }

Output

2024 is a leap year.
